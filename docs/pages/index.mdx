# Starter Template Documentation

Welcome to the Starter Template documentation. This is a turborepo monorepo starter template with the following structure:

## Project Structure

- **apps/web** - Next.js 14 application with TypeScript
- **apps/mobile** - Expo React Native application with TypeScript
- **packages/ui** - Shared component library
- **packages/database** - Supabase types and utilities
- **packages/shared** - Shared utilities, types, and constants
- **docs** - This documentation site

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development servers:
   ```bash
   npm run dev
   ```

3. Build all packages:
   ```bash
   npm run build
   ```

## Available Scripts

- `npm run dev` - Start all apps in development mode
- `npm run build` - Build all apps and packages
- `npm run lint` - Lint all packages
- `npm run test` - Run tests across all packages
- `npm run clean` - Clean all build outputs
